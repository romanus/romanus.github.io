mu, sigma = 1, 1
alpha = 0.95
ns = []
means = []
zCIs_low, zCIs_high = [], []
tCIs_low, tCIs_high = [], []
np.random.seed(1)
for n in range(10, 100000, 1000):
  sample = np.random.normal(mu, sigma, n)

  ns.append(n)
  sample_mean = np.mean(sample)
  sample_variance = math.sqrt(np.var(sample)) # S
  means.append(sample_mean)

  z_low, z_high = st.norm.interval(alpha) # always the same
  z_coeff = sigma / math.sqrt(n)
  zCIs_low.append(sample_mean + z_low * z_coeff)
  zCIs_high.append(sample_mean + z_high * z_coeff)

  t_low, t_high = st.t.interval(alpha = alpha + (1-alpha)/2, df=n-1)
  t_coeff = sample_variance / math.sqrt(n)
  tCIs_low.append(sample_mean - t_high * t_coeff)
  tCIs_high.append(sample_mean + t_high * t_coeff)

Task 1f¶

In the lecture we discussed the CLT for the sample mean. Here it seems to apply to the sample variance too. Why?

Let $X_1$, $X_2$, .. $X_n$ - our samples. Its variance (also Random Variable):

$S^2 = \frac{1}{n-1}\sum_{i=1}^{n} (X_i - \bar{X})^2$

This is a sum of distributions, which tends to have a normal distribution due to the CLT.

Task 5¶

Power of a test: The next aim is to assess the probability of type 2 error (power of a test) of a goodness-of-fit test. Goodness-of-fit tests for the normal distribution are of key importance in statistics, since they allow to verify the distributional assumptions required in many models. Here we check the power of the Kolmogorov-Smirnov test, i.e. is the test capable to detect deviations from normality?

  • Simulate $M = 1000$ samples of size $100$ from a $t$-distribution with $df = 2,...,50$ degrees of freedom. For each sample run the Kolmogorov-Smirnov test and count the cases when the $H_0$ of normality is correctly rejected (for each df). How would you use this quantity to estimate the power of the test? Make an appropriate plot with the df on the X-axis. (Note: the $t$-distribution converges to the normal distribution as df tends to infinity. For $df > 50$ the distributions are almost identical.) Discuss the plot and draw conclusions about the reliability of the test.
In [35]:
n = 100
M = 1000
alpha = 0.05

dfs = np.arange(2, 50)
powers = []

for df in dfs:
  correctly_rejected = 0
  for i in range(M):
    sample = st.t.rvs(df=df, size=n, random_state=df * i)
    statistic, pvalue = st.kstest(sample, "norm", N=n)
    correctly_rejected += int(abs(pvalue) < alpha)
  powers.append(correctly_rejected / M)

The probability of cases when the $H_0$ of normality is correctly rejected is $P(\{H_1\}|H_1)$ - power of a test by definition.

In [36]:
plt.title("Power of a test")
plt.plot(dfs, powers)
plt.xlabel("df")
plt.ylabel("power")
plt.show()

Having small $df$, we correctly reject $H_0$ in 10-25% of cases. As $df$ becomes bigger, we correctly reject $H_0$ in 5-10% of cases. This means that our test has a large Type 2 error, which make it "weak" (non-powerful). This means we don't reject normality in many cases where we should.

Summarising, Kolmogorov-Smirnov test doesn't look like a good option for distinguishing $t$-distributions and normal distributions.
